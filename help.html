Interface

The file is saved every two second if changed with the password in the first field. Passwords have to match, or neither save nor autosave will work.

LoadLoad file with password in first password field.
SaveForce saving the file.
new revisionForce a new revision and save the file.
Clean upDelete all but the current revision.
low contrastSwitches the text to light grey on red, making it harder to read, especially from an angle.
File column (left)Activate file. Create new file by entering name and pressing Create.
Revision column (right)Load specific revision (c = current) with password in first field, generally forcing a new revision when saving.
